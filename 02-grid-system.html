02. Grid System

Bootstrap's grid system uses a series of containers, rows, and columns to layout and align content. It's built with flexbox and is fully responsive.

1. The 12-Column Grid

Bootstrap assumes every row is divided into 12 equal vertical strips (columns). You specify how many strips your content should span.

Rule: Columns (.col-*) must be immediate children of a Row (.row).
.col-4 (1/3 width)
.col-4 (1/3 width)
.col-4 (1/3 width)
.col-8 (2/3 width)
.col-4 (1/3 width)
Bootstrap Way
<div class="row">
  <div class="col-4">...</div>
  <div class="col-8">...</div>
</div>
CSS Flexbox Equivalent
.row {
  display: flex;
  flex-wrap: wrap;
}
.col-4 { width: 33.33%; }
.col-8 { width: 66.66%; }

2. Responsive Columns

You can specify different widths for different screen sizes using breakpoints (sm, md, lg, etc.).

Example: col-12 col-md-6 means "100% wide on mobile, 50% wide on medium screens and up".

(12 on mobile, 6 on desktop)

(12 on mobile, 6 on desktop)

3. Auto Layout Columns

If you use .col without a number, they will share the remaining space equally.
