04. Spacing & Utilities

1. Spacing (Margin & Padding)

Instead of writing margin-top: 1rem;, use mt-3.

Class CSS Equivalent (approx)
m-0 Margin: 0
p-3 Padding: 1rem (16px)
mx-auto Margin-left: auto; Margin-right: auto; (Centers block elements)
my-5 Margin-top: 3rem; Margin-bottom: 3rem;
Visual Example:
p-0
p-3
p-5

Note: I used me-3 (Margin End 3) to separate the boxes.
