08. Images & Media

1. Responsive Images

Use .img-fluid to make an image scale with the parent element.

Note: Since we don't have local images, we are using placeholders.

Fluid Image

Class: .img-fluid (max-width: 100%; height: auto;)

Thumbnail

2. Figures

Use the <figure> component to display images with captions.

Classes: .figure, .figure-img, .figure-caption
