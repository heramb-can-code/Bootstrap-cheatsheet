01. Setup & Containers

What is a Container? In Bootstrap, a container is the most basic layout element. It's required when using the grid system. It centers your content and adds some horizontal padding.

1. Fixed Container (.container)

This container has a max-width at each breakpoint. It sits in the middle of the screen.

Bootstrap Way
<div class="container">Content</div>
Basic CSS Equivalent
.container {
  width: 100%;
  padding-inline: 0.75rem;
  margin-inline: auto;
  max-width: 1320px; /* changes per breakpoint */
}
Result Demo:
I am inside a .container. Resize your browser window to see me snap to specific widths!

2. Fluid Container (.container-fluid)

This container is always 100% wide, regardless of screen size.

Bootstrap Way
<div class="container-fluid">Content</div>
Basic CSS Equivalent
.container-fluid {
  width: 100%;
  padding-inline: 0.75rem;
  margin-inline: auto;
  /* No max-width restriction */
}
Result Demo:
I am inside a .container-fluid. I will always stretch to the edges.

3. Breakpoints (Cheat Sheet)

Bootstrap uses these default breakpoints for its media queries.

Breakpoint Class Infix Dimensions
X-Small None <576px
Small sm ≥576px
Medium md ≥768px
Large lg ≥992px
Extra Large xl ≥1200px
XXL xxl ≥1400px
